Research Phases

Phase 1: Theory
Phase 2: VR Lab
Phase 3: Bio-Robotics

Vicsek Model Extension

Extend classical collective behavior models to include flow field interactions and multi-sensory integration.

  • Flow gradient analysis
  • Turbulence effects modeling
  • Large-scale current integration

Cognitive Integration

Build on Sayin et al. (2025) findings to create flow-aware decision-making models.

  • Visual-flow information fusion
  • Probabilistic decision frameworks
  • Multi-scale temporal dynamics

VR System Design

Create immersive virtual reality environment for collective behavior experiments.

  • Multi-agent interaction protocols
  • Real-time flow visualization
  • Haptic feedback integration

Flow Laboratory

Validate virtual experiments with physical flow measurements.

  • Particle tracking velocimetry
  • Wind/water tunnel validation
  • Sensor calibration protocols

Mudskipper-Inspired Swarms

Develop amphibious robotic swarms based on bio-inspired locomotion principles.

  • Air-water transition mechanisms
  • Distributed coordination algorithms
  • Environmental adaptation

Flow-Sensing Networks

Implement flow-aware collective intelligence in robotic systems.

  • Pressure array sensors
  • Accelerometer integration
  • Real-time swarm coordination
